-> Configure o PC com nome e email do responsável pelo código 
git config --global user.name "Eduardo Petarnella Gabri"
git config --global user.email eduardo1227edu@gmail.com

-> Navegando no CMD
cd nome da pasta  //entre em uma pasta
cd..              //volte uma pasta 
cd /              //vai para o c:

-> inicializa um repositorio git (utilizado apenas uma vez)
git init

-> Verificar o status do repositório
git status

-> Adicionar os novos arquivos no git para versionar
git add.

-> Salva uma versão dos arquivos
git commit -m "Porque estou salvando???"

-> Envia informação do PC para o GitHub
git push 

-> baixa a atualização dos arquivos do github para uma pasta já existente 
git pull
